# Blu & Claude's Side-Scroller Adventure

A Mario-style platformer game built with Phaser.js and Vite.

## Features
- 3 progressively challenging levels
- Smooth player mechanics (jump, crouch, movement)
- Enemy AI (ground patrol and flying enemies)
- Power-ups (coins, stars, mushrooms)
- Polished HUD with heart-based health display
- Complete game flow with victory and game over screens

## Controls
- **Arrow Keys**: Move left/right
- **Up Arrow**: Jump
- **Down Arrow**: Crouch
- **ESC**: Return to menu

## Technologies
- Phaser.js 3.80.1
- Vite 5.4.11
- JavaScript (ES6+)

## Development
```bash
npm install
npm run dev
```

## Build for Production
```bash
npm run build
```
